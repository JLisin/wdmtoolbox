'wdmtoolbox' Documentation
==========================

This is the home page for 'wdmtoolbox'.  'wdmtoolbox' is a command line program to work with Watershed Data Management (WDM) files used to store input and output time-series data for the Hydrological Simulation Program - FORTRAN (HSPF).

'wdmtoolbox' will work with Python 2.6+ and 3.0+.
